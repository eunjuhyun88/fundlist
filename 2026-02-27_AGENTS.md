# VCList / fundlist Agent Guide (Compact)

## Project Scope
- Primary app: `fundlist` (this repository).
- Goal: collect investment data, store to SQLite, return list views.
- Default work target: files under `src/fundlist/`.

## Stack
- Python 3 (stdlib-first; avoid new deps unless user asks).
- SQLite (`data/investment_items.db` by default).
- External APIs: GitHub, SEC EDGAR, FRED, CoinGecko.

## Commands
```bash
# From repo root
python3 fundlist.py collect
python3 fundlist.py list --limit 20
python3 fundlist.py ops-sync --alert-days 14
python3 fundlist.py ops-list --from-days -365 --to-days 30
python3 -m py_compile fundlist.py src/fundlist/*.py src/fundlist/sources/*.py
```

## Data Contract
- Table: `investment_items`
- Required fields:
  - `source`, `category`, `symbol`, `title`, `url`, `published_at`, `payload_json`
  - `fingerprint` (unique dedupe key)

## Implementation Rules
- Keep connectors isolated per source file.
- Normalize all external records into `Item`.
- Never put secrets/API keys in code or logs.
- Fail one connector without failing the whole collect flow.
- Keep output stable: `collect` summary + `list` pipe-delimited rows.

## Context Budget Rules
- Read this file first; open only needed modules.
- Use `rg` to jump directly to symbols instead of full-file reads.
- Use `docs/ARCHITECTURE.md` only when structural decisions are required.
- Avoid loading DB payload blobs unless debugging data quality.

## Context Control Commands
- `현재를 저장`:
  - Generate a short project-state summary from current chat/work.
  - Run `python3 scripts/context_ctl.py save --label manual --summary "<summary>"`.
- `컨텍스트 컴팩션 해`:
  - Run `python3 scripts/context_ctl.py compact`.
  - Use `.context/COMPACT.md` as the default recovery summary.
- `복구해줘`:
  - Ask first: context restore vs file/code restore.
  - If context restore, run `python3 scripts/context_ctl.py restore --mode compact`.
- `주기적으로 컴팩션`:
  - Suggest automation/cron schedule before enabling.
  - Default recommendation: every 3 hours during active work sessions.
  - Cron target command: `/bin/zsh "/Users/ej/Downloads/VC list/fundlist/scripts/context_cron.sh"`.
